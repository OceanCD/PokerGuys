# PokerGuys 🃏

Texas Hold'em poker session tracker & finance manager.

## Features

- **Session Recording**: Track buy-ins and final stacks for each player
- **Balance Validation**: Auto-check if table is balanced (net zero sum)
- **Player Stats**: Lifetime P&L tracking per player
- **Clean UI**: Binance-inspired dark/light theme
- **Local Storage**: SQLite database (no cloud needed)

## Quick Start

```bash
# Install dependencies
pip install -r requirements.txt

# Run the app
streamlit run main.py
```

## Tech Stack

- **Frontend**: Streamlit (Python)
- **Database**: SQLite
- **Charts**: Plotly
